🚇 MetroPulse: Delhi Metro Travel Analytics

📖 Project Overview
This project analyzes metro travel data using Python and SQL to generate insights on passenger behaviour, revenue trends, and travel patterns.

🎯 Objectives
	•	Clean and prepare metro trips data using Python
	•	Store the dataset in a structured SQL database
	•	Perform analytical queries using SQL
	•	Analyse passenger travel behaviour
	•	Identify high-demand routes and station
	•	Study revenue generation across trips
	•	Understand ticket usage patterns
	•	Evaluate travel conditions such as peak, festival and weekend trips

📊 Dataset Description
The dataset contains metro trip-level data capturing passenger travel information, routes, fares, and travel conditions.
Column Name
Description
TripID
Unique identifier for each metro trip
Date
Date of travel
From_Station
Origin metro station
To_Station
Destination metro station
Distance_km
Distance between stations
Fare
Total fare collected
Cost_per_passenger
Average cost per passenger
Passengers
Number of passengers traveling
Ticket_Type
Type of ticket used
Remarks
Travel condition (Peak, Off-peak, Festival, Weekend, Maintenance)
